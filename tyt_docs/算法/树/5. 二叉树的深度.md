# 二叉树的深度

## 基础概念
- **二叉树的深度：** 指从根节点到该节点的最长简单路径边的条数。
- **二叉树的高度：** 指从该节点到叶子节点最长简单路径边的条数。

## 二叉树的最大深度

### 递归法 - 后序遍历
```python
class Solution:
      def maxDepth(self, root):
        """
        :type root: Optional[TreeNode]
        :rtype: int
        """
        if not root: return 0
        leftDeep = self.maxDepth(root.left)
        rightDeep = self.maxDepth(root.right)
        return 1 + max(leftDeep, rightDeep)
```
### 队列法-层序遍历
```python
class Solution(object):
    def maxDepth(self, root):
        """
        :type root: Optional[TreeNode]
        :rtype: int
        """
        if not root: return 0
        depth = 0
        queue = deque([root])
        while queue:
          depth += 1
          for _ in range(len(queue)):
            curPop = queue.popleft()
            if curPop.left: queue.append(curPop.left)
            if curPop.right: queue.append(curPop.right)
        return depth
```

## 二叉树的最小深度

### 递归法 - 后序遍历
```python
class Solution:
      def minDepth(self, root):
        """
        :type root: Optional[TreeNode]
        :rtype: int
        """
        if not root: return 0
        leftDeep = self.minDepth(root.left)
        rightDeep = self.minDepth(root.right)
        if root.left and not root.right:
            return 1 + leftDeep
        if root.right and not root.left:
            return 1 + rightDeep
        return 1 + min(leftDeep, rightDeep)
```
### 队列法-层序遍历
```python
class Solution(object):
    def minDepth(self, root):
        """
        :type root: Optional[TreeNode]
        :rtype: int
        """
        if not root: return 0
        depth = 0
        queue = deque([root])
        while queue:
          depth += 1
          for _ in range(len(queue)):
            curPop = queue.popleft()
            if curPop.left: queue.append(curPop.left)
            if curPop.right: queue.append(curPop.right)
            if not curPop.left and not curPop.right:
              return depth
        return depth
```

## 最大深度和的最小深度的区别
- 主要在于处理左右孩子不会空的逻辑
- 最小深度是从根节点到最近叶子节点的最短路径上的节点数量。注意是叶子节点
- 最大深度是从根节点到最远叶子节点的节点数量。
