# 词嵌入
## 基本概念
词嵌入是 “把文字转成数字向量” 的操作，转出来的这个数字向量，就是词向量。
**词嵌入（Embedding）= 词向量（Word Vector）的「生产过程 + 最终产物」**

## 为什么需要用词嵌入/词向量
计算机只认识数字（整数 / 浮点数），但我们的文本是文字（比如 “苹果”“电脑”“喜欢”），直接喂给模型（大模型 / 机器学习模型），模型根本无法处理。
比如想让模型判断 “苹果” 和 “香蕉” 更像，还是和 “电脑” 更像，文字本身比不了，但转成数字后，就能通过计算数字的 “相似度” 判断。这就是词嵌入 / 词向量的核心作用。

**词向量和词嵌入对比**
- 词向量：结果，是文字的 “数字身份证”，是一串有含义的浮点数列表，核心作用是「让文字可计算」；
- 词嵌入：过程 / 技术，是 “制作数字身份证的方法”，核心作用是「把文字转成词向量」，也可以指代模型中负责做这个转换的「嵌入层（Embedding Layer）」。
